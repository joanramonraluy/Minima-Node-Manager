Minima Help
-data               Specify the data folder ( defaults to ~/.minima )
-dbpassword         Main Wallet / SQL AES password - MUST be specified on first launch. CANNOT be changed later.
-basefolder         Specify a default file creation / backup / restore folder
-host               Specify the host IP
-port               Specify the Minima port
-rpc                Specify the RPC port
-rpcenable          Enable rpc     
-rpcpassword        Set Basic Auth password for RPC calls ( Use with SSL / stununel )
-rpcssl             Use Self Signed SSL cert to run RPC
-rpccrlf            Use CRLF at the end of the RPC headers (NodeJS)
-shownetcalls       Show all the network calls
-shownetcallsnopoll Show all the network calls except poll messages
-allowallip         Allow all IP for Maxima / Networking
-archive            Run an Archive node - store all data / cascade for resync
-mdsenable          Enable MDS     
-mdspassword        Specify the Minima MDS password
-mdsinit            Specify a folder of MiniDAPPs
-mdswrite           Make an initial MiniDAPP WRITE access
-conf               Specify a configuration file (absolute)
-daemon             Run in daemon mode with no stdin input ( services )
-isclient           Tells the P2P System that this node can't accept incoming connections
-desktop            Use Desktop settings - this node can't accept incoming connections
-server             Use Server settings - this node can accept incoming connections
-mobile             Sets this device to a mobile device - used for metrics only
-jnlp               Are we running from the JNLP
-showparams         Show startup params on launch
-nop2p              Disable the automatic P2P system
-noshutdownhook     Do not use the shutdown hook (Android)
-noconnect          Stops the P2P system from connecting to other nodes until it's been connected to
-p2prootnode        Specify the initial P2P host:port to connect to
-p2pnodes           Specify a list of nodes (or an URL to a file with the list) to use IF your peers list is empty
-p2p-log-level-info Set the P2P log level to info
-p2p-log-level-debugSet the P2P log level to debug
-p2p2               Enable the new P2P2 system
-connect            Disable the p2p and manually connect to this list of host:port
-clean              Wipe data folder at startup
-nodefaultminidapps Do NOT install the default MiniDAPPs
-nosyncibd          Do not sync IBD (for testing)
-syncibdlogs        Show detailed SYNC_IBD logs
-megammr            Are we running in MEGA MMR mode
-notifyalltxpow     Send notification messages for ALL TxPoW (not just relevant)
-slavenode          Connect to this node only and only accept TxBlock messages.
-limitbandwidth     Limit the amount sent for archive sync
-genesis            Create a genesis block, -clean and -automine
-test               Use test params on a private network
-solo               Run a solo/private network (-test -nop2p) and will run -genesis ONLY the first time
-testchainlength    Specify length of tree to keep in -test mode (default is 32)
-nosslmds           Disable the self signed SSL cert for MDS. Use STunnel or your own SSL proxy instead.
-mysqldb            Set the full MySQL DB details as username:password@host:port
-mysqldbcoins       Enable the MySQL coins db backup from CLI
-mysqldbdelay       When running in Docker.. Delay in milli-seconds before attempting first MySQL Connection
-mysqlalltxpow      Store all TxPoW in MySQL when autobackup enabled.
-txpowdbstore       How many days to store TxPoW in the internal H2 Database (default 3)
-publicmds          Enable the Public MDS system from the CLI
-publicmdsuid       Set your own Session ID for the Public MDS and enable Public MDS
-rescuenode         If you connect to a heavier chain use this MegaMMR node to resync
-help               Print this help
-seed               Use this seed phrase if starting a new node
-megaprune          Prune unspendable addresses from the megammr
-megaprunestate     Prune all coins with a state (useful for exchanges)
-megaprunetokens    Prune all tokens - only keep Minima coins
