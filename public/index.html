Web / MiniDapp Management

Install / Update: Smart deployment that automatically detects if the MiniDapp is already installed on each **active node**. Performs a fresh install if missing, or an update (preserving data) if present.
Push to Mobile: Installs or updates the MiniDapp within Minima on the **Target ADB Device** selected in the section below.

Android Management (ADB)

Target ADB Device

Install / Update APK: Uploads and reinstalls the APK on the device. Android will treat this as an update if the app is already present.
Uninstall APK: Completely removes the application from the device.

Build Pipeline (Source to Artifact)

Build & Zip: Compiles the web app (`npm run build`) and creates the MiniDapp package (`.mds.zip`).

Build Android APK: Compiles the project and generates the debug APK.

Build All (Zip + APK): The complete build pipeline. Generates both the `.mds.zip` and the Android APK.
